# CoinFlipWithBet – Soroban Smart Contract

## Project Description

CoinFlipWithBet is a decentralized betting smart contract built on the Stellar blockchain using the Soroban smart contract platform.  

The contract allows two players to participate in a simple coin flip betting game. Each player places a bet, and the winner is randomly selected through a pseudo-random mechanism. The winner receives the total bet amount.

This project demonstrates how decentralized gaming and betting applications can be implemented on Stellar using Soroban smart contracts.

---

## What It Does

1. Player 1 starts the game and places a bet.
2. Player 2 joins the game with the same bet.
3. The contract performs a coin flip using a pseudo-random method.
4. One player is selected as the winner.
5. The winner receives the total bet amount.

---

## Features

- Decentralized betting system
- Built on the Stellar Soroban smart contract framework
- Transparent on-chain logic
- Simple two-player betting game
- Automatic winner selection
- Immutable smart contract execution
- Lightweight and gas-efficient
